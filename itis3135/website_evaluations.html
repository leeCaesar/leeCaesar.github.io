CRAP Website

This website is one that will show you an example of a good website and one of a bad website from my standards Based off of what I learned on the principles of Crap
Crap Is an acronym in which the

  • C stands for Contrast
    • which means that your main color schemes should not have the same brightness of color.
    For example if you were to use a light color blue for a background, the text on top should be closer to that
    of dark red or black. The same is the case for using bright colors.
  • R stands for Repetition
    • which means that your font styles and sizes should not vary to much in size and type. The same applies for colors
    shapes and even line thickness.
  • A would stand for Alignment
    • which would mean that your page should be aligned with an axis, and this applies to all contents in the page.
  • P would stand for proximity
    • which means that objects that are related to each other should not only be close to each other on a page seperating
    with a line or font size makes your page easier to read.
When all of these elements of design come together will have a bad website design thats not easy to read and will be unattractive to visitors.
The "Bad Website" link at the top of the page links to a website with not only terrible alignment, the color scheme for fonts is all over the place,
and none of the photos were formated to a nice size so you have photos all over the place with hard to read descriptions and sections of the page bleeding into the margins of the page.
The "Good Website" link at the top of that page links to the twitter homepage which I would consider a good design.
